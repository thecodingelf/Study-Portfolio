Responsive Web Design (RWD)

Published: 2017-01-17

It offers users user friendly enviroment depending what device they are using. To create smooth and nice websites for all screen sizes and is easy to adept to new screen sizes and devices.

Before responsive design designers didn't take it account how websites did look on the mobile devices since they were new and it took time to start also developing the websites to mobile.

Difference now and then is that websites are looking smooth and designed really well to fit for all devices and offer best user interface experience to the users.

Media Queries

Published: 2017-01-17

Media queries will help you to build responsive websites that are scaling to the different devices like mobile phones and tablets. You can also determine them in css file which causes same effect as used in html file.
