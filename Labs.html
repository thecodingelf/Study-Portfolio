Definitions

  • HTML: It is basic language that can be used to build websites and web applications and it is base structure for all websites.
  • HTML5: It is current version of the HTML language which brought new features for handling multimedia and graphic content it also has new structure elements that can be used to create websites.
  • CSS: It is cascade stylesheet that is used to create visual look to the websites.
  • HTTP: It is an application protocol that is used to transfer data.
  • Javascript: It is a script language that can be used to change date automatically in website.
  • Dynamic HTML (DHTML): It is combination of languages that can be used together to create content in websites and change effects in the website. You can also create games with dynamic HTML but it is not commonly used method to create games.

Responsive Web Design (RWD)

Published: 2017-01-17

It offers users user friendly enviroment depending what device they are using. To create smooth and nice websites for all screen sizes and is easy to adept to new screen sizes and devices.

Before responsive design designers didn't take it account how websites did look on the mobile devices since they were new and it took time to start also developing the websites to mobile.

Difference now and then is that websites are looking smooth and designed really well to fit for all devices and offer best user interface experience to the users.
